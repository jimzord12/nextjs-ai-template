# Domain Book

The Domain Book is the authoritative, persistent record of domain understanding for this project. It captures the concepts, rules, relationships, boundaries, states, decisions, risks, and open questions that define the system.

Each chapter has its own directory under `book/chapters/`. Content is built incrementally through the promotion of completed Single Units of Work (SUWs) from the Milestones system.
