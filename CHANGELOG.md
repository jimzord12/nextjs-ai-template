## [0.1.0] - 2026-06-07

### Added

- Initial Next.js 16 template scaffold with TypeScript strict mode
- Biome for linting and formatting
- Vitest for unit testing
- react-doctor score gate in CI (threshold ≥ 80)
- GitHub Actions CI pipeline (lint → test → build → audit → doctor)
- Husky pre-commit hooks with lint-staged
- pnpm as enforced package manager (v11.5.2)
- Issues manager CLI tool for local issue and feature tracking
- i18n routing with next-intl and locale message files (en, de, el)
- Tailwind CSS 4 integration with tw-animate-css
- shadcn/ui component library setup
- Agent skills and AGENTS.md conventions for AI-assisted development
- PR template and GitHub Flow with branch protection on main
- Project documentation: ARCHITECTURE.md, CONVENTIONS.md, WORKING_WITH_ME.md, RULES.md
- QA scripts: performance, accessibility, security audit, and quality gates
- Unlighthouse integration for performance auditing
